#  イヌネコ判定機

Microsoftが提供するイヌネコの画像セットをCreateMLで学習させたモデルを使って、選択した画像が🐶か😺かを判定します。

データセットは[こちら](https://www.microsoft.com/en-us/download/details.aspx?id=54765)からダウンロードできます。
